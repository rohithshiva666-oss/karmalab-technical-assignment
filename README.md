# Karmaa Lab – Full Stack Task Manager

A simple full-stack Task Manager application built as part of the Karmaa Lab technical assignment.

The application supports full CRUD operations for tasks with a React frontend and a Node.js + Express backend.

---

# Tech Stack

## Frontend
- React (Vite)
- JavaScript (ES6)
- CSS (custom styling)
- Fetch API

## Backend
- Node.js
- Express.js
- In-memory data storage (no database)




---

## Features

- Create new tasks
- View all tasks
- Update task title
- Toggle task completion
- Delete tasks
- Responsive UI
- Clean REST API structure

## How to Run the Project Locally

### 1. Clone the repository

git clone https://github.com/rohitshiva666-oss/karmalab-technical-assignment.git
cd karmalab-technical-assignment

##Run Backend
cd backend
npm install
node index.js

## Run Front end
cd frontend
npm install
npm run dev

## frontend server
http://localhost:5173

## backend server
http://localhost:5000
